Nano Banana 有两种用法：
1. **从零生成**：写完整的构图、风格、氛围 prompt（如之前的黑白封面 prompt）
2. **针对性调整**：上传已有图片，prompt 只写需要改动的那一点，其余全部不提

**Why:** 用户纠正了我之前的误解——Nano Banana 可以从零生成，但更擅长小幅迭代调整。调整时只需说明改动点，不需要重写整个 prompt，重写反而会破坏原图效果。

**How to apply:**
- 调整场景：prompt 极简，只描述改动。例如"把狗的脸转向右侧"、"给图片上色，用暖琥珀色打亮狗和衣摆，背景用冷靛蓝"
- 不要在调整 prompt 里重复风格、构图、情绪等原图已有的描述
- 생成新图时才写完整 prompt
